# BudgetBuddy

A budgeting tool designed for integration into online banking apps. BudgetBuddy helps you create and stick to your budget using AI-powered insights.

## Overview

BudgetBuddy transforms personal finance management by combining intelligent budgeting with behavioral incentives. The application analyzes your spending patterns and provides actionable recommendations to help you achieve your financial goals while making budgeting engaging through a streak-based reward system.

## Features

### Budget Management
- Create custom budgets with personalized spending allocations across categories
- Set monthly spending limits and track progress in real-time
- Flexible budget updates and adjustments based on changing needs

### AI-Powered Insights
- Gemini Flash integration provides intelligent budgeting suggestions
- Automated spending analysis and category-based recommendations
- Smart cost-splitting guidance to stay under budget goals

### Streak System
- Track consecutive days or weeks of staying under budget
- Gamified experience encourages consistent financial discipline
- Enables banks to reward users with sponsor discounts, cashback, or loyalty points

### Transaction Management
- Comprehensive transaction tracking and history
- Automatic categorization of expenses
- Visual spending breakdowns and analytics

### Notifications
- Real-time alerts when approaching budget limits
- Weekly progress summaries
- Streak milestone notifications

## Tech Stack

- Next.js with TypeScript
- Gemini Flash AI
- MongoDB
- Tailwind CSS

## Use Cases

BudgetBuddy is designed for financial institutions looking to enhance their digital banking platforms with advanced budgeting capabilities. By integrating BudgetBuddy, banks can increase user engagement, promote financial wellness, and build stronger customer relationships through rewards programs tied to responsible spending habits.
